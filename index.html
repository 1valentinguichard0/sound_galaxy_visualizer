// Concept

Ta bibliothèque devient un univers

Tout tes artistes favoris réunis dans une galaxie musicale. Chaque planète porte la couleur de l'artiste, tourne sur elle-même et brille selon le nombre de titres dans ta collection.

// Comment exporter depuis Apple Music
1. Ouvre Musique sur Mac
2. Menu Fichier → Bibliothèque → Exporter la bibliothèque…
3. Sauvegarde le fichier .xml sur ton bureau
4. Glisse-dépose le fichier ci-dessous
// Comment exporter depuis Spotify
1. Va sur exportify.net dans ton navigateur
2. Connecte ton compte Spotify
3. Clique Export All pour télécharger un .csv par playlist
4. Glisse-dépose un des fichiers .csv ci-dessous
// Comment exporter depuis Deezer
1. Va sur deezer.com → Paramètres → Mon compte
2. Section Télécharger mes données → Demander l'export
3. Reçois le .zip par email, extrais le fichier .csv
4. Glisse-dépose le fichier ci-dessous
// Comment exporter depuis SoundCloud
1. Va sur soundiiz.com et connecte ton compte SoundCloud
2. Dans le menu gauche, clique sur SoundCloud → Liked Tracks
3. Clique Export → choisis le format CSV
4. Glisse-dépose le fichier .csv ci-dessous
